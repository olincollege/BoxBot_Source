# BoxBot_Source
Source code for Box Bot PIE project @Olin College. Includes all code that is run on the robot (in robot folder), as well as the laptop code (in laptop folder).

## Disclaimer
This code represents the current status of the Box Bot project at the end of the Fall 2025 semester, and is in an incomplete state. Given the short timeline and ambitious scope of the Box Bot project, the authors used generative AI tools to assist with code creation and de-bugging.
